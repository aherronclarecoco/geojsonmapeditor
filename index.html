GeoJSON Station Editor

Step 1: Upload your GeoJSON file. If you don't upload a file, the default file containing all the fire stations in Ireland will be used.

Step 2: Enter the station names (one per line) and the properties you want to update.

Step 3: Click "Check Station Names" to verify the names.

Step 4: If all names are correct, choose to update the GeoJSON or hide unedited pins.









Using geojson.io

geojson.io is a tool for creating, editing, and visualizing GeoJSON files. Follow these steps:

  1. Go to geojson.io.
  2. Upload the GeoJSON file you generated.
Using smappen

smappen is a tool for creating custom maps and exporting them as GeoJSON files. Follow these steps:

  1. Go to smappen.com/app.
  2. Upload the CSV you generated, select co-ordinates and you can calculate the travel times from there.
